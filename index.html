MidJourneyCN 中文汉化插件

一款专为 MidJourney 用户打造的中文优化工具，解锁更便捷的 AI 绘图体验。

插件亮点 ✨

📌 全面汉化

支持 MidJourney V6/V7 的核心功能模块，含评分、编辑器、变体等。

📋 实用功能

支持界面翻译、关键按钮优化、使用流程汉化，提升使用体验。

🧩 适配平台

支持 Chrome、Edge 和 Safari（Mac）等主流浏览器，安装灵活，跨平台使用。

🔧 轻量高效 / 完全免费

插件轻量安装，开箱即用。

安装方式 📦

🧭 Chrome 浏览器安装步骤

  1. 访问 Releases 页面 下载最新插件 .zip 文件
  2. 解压压缩包
  3. 打开 chrome://extensions/
  4. 开启右上角「开发者模式」
  5. 点击「加载已解压的扩展程序」，选择解压后的插件文件夹

🧭 Edge 浏览器安装步骤

  1. 访问 Releases 页面 下载最新插件 .zip 文件
  2. 解压压缩包
  3. 打开 edge://extensions/
  4. 开启右上角「开发者模式」
  5. 点击「加载已解压的扩展程序」，选择解压后的插件文件夹

🧭 Safari 浏览器（Mac）安装方法

  1. 访问 Releases 页面 下载最新 MidjourneyCN for Mac 安装包
  2. 打开 App 并点击“打开 Safari 插件设置”
  3. Safari → 设置 → 扩展 → 启用插件
  4. 如无法启用：
    • Safari → 设置 → 高级 → 勾选“在菜单栏显示‘开发’”
    • 菜单栏 → 开发 → 勾选“允许未经签名的扩展”
